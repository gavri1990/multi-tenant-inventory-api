# Multi-Tenant Inventory API

A modern REST API project showcasing **cloud-native development** with **multi-tenant architecture**, **role-based access control**, and **JWT authentication**. The goal is to demonstrate scalable design patterns and deployment practices using **Azure hosting**, **Docker**, and a CI/CD pipeline.
